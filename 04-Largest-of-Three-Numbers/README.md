# Problem 04 - Largest of Three Numbers

## Problem Statement
Write a Python program to find the largest of three numbers.

## Example
Input:
10
25
15

Output:
25 is the largest number.

## Logic
Compare all three numbers.
If the first number is greater than or equal to the other two, it is the largest.
Otherwise, check the second number.
If neither the first nor the second is the largest, then the third number is the largest.
