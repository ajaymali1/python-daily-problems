# Problem 09 - Multiplication Table of a Number

## Problem Statement

Write a Python program to print the multiplication table of a given number.

## Example
Input:
5

Output:
5 x 1 = 5
5 x 2 = 10
...
5 x 10 = 50

## Logic
- Take a number as input.
- Use a loop from 1 to 10.
- Multiply the number by each value in the loop.
- Print the result in table format.
